# Security Policy

## Reporting a vulnerability

**Please do not open public GitHub issues for security vulnerabilities.**

Report privately to **security@lavasecurity.app** _(TODO: confirm address / enable
GitHub private vulnerability reporting)_. Include:

- A description of the issue and its impact.
- Steps to reproduce or a proof of concept.
- The app version / commit and iOS version.

We aim to acknowledge reports within a few business days and will keep you
updated on remediation. Please give us a reasonable window to ship a fix before
any public disclosure.

## Scope

This repository is the iOS client. Backend and infrastructure issues should also
be reported to the address above; they are handled in private repositories.
